# Installation

## From PyPI

```bash
pip install logodev-mcp
```

## From Docker

```bash
docker pull ghcr.io/pvliesdonk/logodev-mcp:latest
```

## From source

```bash
git clone https://github.com/pvliesdonk/logodev-mcp
cd logodev-mcp
uv sync --all-extras --all-groups
```
